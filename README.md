# Echo app in Ruby

## Description
A simjple echo application to echo back the date with what you said

## Tech Stack
Ruby
RSpec

## Installation

    $ git clone 
    $ cd echo_app_rb
    $ open ./lib/echo_app.rb
    $ rspec #Run the test first

## How to use
    $ irb
    $ require('./lib/echo_app.rb')
    $ echo('this is an echo app')
